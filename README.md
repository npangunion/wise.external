# [wise.external] 외부 도구  

wise.external은 wise 프로젝트가 사용하는 외부 라이브러리를 두는 곳이다. 

- boost
  - 모든 C++ 프로젝트들 사용 
- premake5 
  - 모든 C++ 프로젝트들 필수 사용
- opencppcoverage
  - https://marketplace.visualstudio.com/items?itemName=OpenCppCoverage.OpenCppCoveragePlugin
  - 코드 커버리지 툴 
  - VS 2019 지원 
- libsodium 
  - 암호, 해시 등 
  - 크로스 플래폼 
  - 8K star 

- spdlog 
  - 매우 빠르고 편리한 c++ 로깅 라이브러리 
- catch 
  - c++ unittest 라이브러리
